# Жемчужины Беларуси

Интерактивная веб-страница, представляющая четыре ключевые достопримечательности Беларуси:
- Брестская крепость
- Мирский замок
- Несвижский замок
- Беловежская пуща

## Особенности
- Современный адаптивный дизайн
- Анимации при прокрутке (Scroll Reveal)
- Интерактивный фон со звездным небом
- Прямые переходы на статьи в Википедии

## Развертывание
Проект готов к публикации на GitHub Pages. Главный файл переименован в `index.html` для автоматического подхвата сервером.
